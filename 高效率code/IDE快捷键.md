目前使用的ide或者编辑器主要有3个 vs,vs code qtcreator。

## 通用
1. 全选 Ctrl + A
2. 复制 Ctrl + C
3. 粘贴 Ctrl + V
4. 剪贴 Ctrl + X
5. 撤销 Ctrl + Z 反撤销 Ctrl + Y
6. 注释和取消注释 Ctrl+/ 
7. 打开关闭资源管理器(侧边栏)，Ctrl + Shift + E 
## QtCreator
1. Ctrl + K focuse 快速打开小窗口
2. Ctrl + F5 运行
3. Ctrl + 2  文本编辑窗口
4. Ctrl + 6 进入帮助文档
5. 
6. F1 查看选中的文档
